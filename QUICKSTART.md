# MyBank - Быстрый старт

## 🚀 Инструкция по запуску

### Шаг 1: Получить токен бота

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. **Скопируйте токен** (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Установить токен

**Windows PowerShell:**
```powershell
$env:TELEGRAM_BOT_TOKEN="ваш_токен_здесь"
```

**Windows CMD:**
```cmd
set TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

### Шаг 3: Запустить бота

```bash
# Активировать окружение
.venv\Scripts\activate

# Запустить
python bot.py
```

### Шаг 4: Открыть бота в Telegram

1. Найдите своего бота в Telegram (по имени, которое вы дали)
2. Нажмите **Start** или отправьте `/start`
3. Готово! 🎉

## 💰 Как добавить тестовый баланс

```bash
# Формат: python add_balance.py [telegram_id] [сумма]

# Узнать свой Telegram ID:
# - Напишите боту @userinfobot
# - Он покажет ваш ID

# Пример: добавить 1000 рублей
python add_balance.py 123456789 1000
```

## 📱 Доступные команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу и регистрация |
| `/balance` | Проверить баланс |
| `/transfer` | Перевести деньги |
| `/history` | История транзакций |
| `/help` | Справка |
| `/cancel` | Отменить операцию |

---

## 🔄 Как перевести деньги

1. Отправьте `/transfer`
2. Введите **ID** или **username** получателя
3. Введите **сумму** (например: `100.50`)
4. Добавьте **описание** или напишите `-` для пропуска
5. Готово! ✅

## ⚙️ Для продвинутых

### Изменить путь к базе данных

```bash
$env:DATABASE_PATH="c:\\path\\to\\mybank.db"
```

### Режим логирования

Логи выводятся в консоль. Для отладки измените в `config.py`:
```python
LOG_LEVEL = "DEBUG"  # Вместо "INFO"
```

## ❓ Troubleshooting

**Бот не запускается:**
- Проверьте, что токен установлен правильно
- Убедитесь, что зависимости установлены: `uv pip install -r requirements.txt`

**"User not found":**
- Сначала отправьте `/start` боту

**Перевод не проходит:**
- Проверьте достаточность средств
- Убедитесь, что получатель существует
