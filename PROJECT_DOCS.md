# MyBank - Проект онлайн-банка для Telegram

## 📋 Информация о проекте

### Конечный пользователь
Владельцы смартфонов с Telegram, которым нужен быстрый и простой способ:
- Отслеживать свой баланс
- Переводить деньги другим пользователям
- Просматривать историю транзакций

### Какую проблему решает продукт
Упрощает банковские операции для небольших переводов между пользователями Telegram без необходимости заходить в полноценное банковское приложение.

### Идея продукта в одном предложении
MyBank - это Telegram-бот для мгновенных переводов между пользователями с отслеживанием баланса и истории транзакций.

### Ключевая функция
**Переводы между пользователями Telegram** - быстро, просто, без комиссии.

## 🚀 План реализации

### Version 1 - MVP (Текущая версия)

**Статус:** ✅ Готово к демонстрации

**Реализованные возможности:**
- ✅ Регистрация пользователей при первом запуске
- ✅ Проверка баланса
- ✅ Переводы денег между пользователями (по ID или username)
- ✅ История транзакций (отправленные и полученные)
- ✅Inline-клавиатура для удобной навигации
- ✅ SQLite база данных
- ✅ Валидация сумм и лимиты
- ✅ Обработка ошибок

**Компоненты:**
1. **Backend:** Python + SQLAlchemy ORM
2. **Database:** SQLite (mybank.db)
3. **Client:** Telegram Bot (python-telegram-bot)

**Как запустить:**
```bash
# 1. Установить зависимости
uv pip install -r requirements.txt

# 2. Установить токен бота (Windows PowerShell)
$env:TELEGRAM_BOT_TOKEN="your_token_from_botfather"

# 3. Запустить бота
.venv\Scripts\activate
python bot.py
```

**Команды бота:**
- `/start` - Регистрация и приветствие
- `/balance` - Проверить баланс
- `/transfer` - Перевести деньги
- `/history` - История транзакций
- `/help` - Справка
- `/cancel` - Отменить операцию

## 🎯 Как продемонстрировать TA

### Для Version 1:

1. **Запустить бота:**
   ```bash
   $env:TELEGRAM_BOT_TOKEN="your_token"
   python bot.py
   ```

2. **Открыть бота в Telegram и показать:**
   - `/start` - регистрация пользователя
   - `/balance` - проверка баланса
   - Добавить баланс через `python add_balance.py <telegram_id> 1000`
   - `/transfer` - перевод другому пользователю
   - `/history` - просмотр истории транзакций

3. **Показать код:**
   - Структуру базы данных
   - Логику переводов
   - Обработку ошибок
